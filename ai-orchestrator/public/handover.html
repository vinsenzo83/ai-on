⚠️주의사항 & 알려진 이슈
🔑
샌드박스 vs VPS 환경 차이
샌드박스에서는 일부 API 호출이 WAF/Cloudflare로 차단되거나 응답이 왜곡될 수 있습니다. 반드시 VPS 실서버 기준으로 provider 상태를 재확인하세요.
💾
SQLite WAL 체크포인트 미실행
orchestrator.db-wal이 1.2MB로 커졌습니다. 장기 운영 시 WAL이 계속 커지므로 주기적 체크포인트 권장: PRAGMA wal_checkpoint(TRUNCATE); 또는 재시작 시 자동 처리됨.
💰
costs 테이블 비어있음 (0건)
inference_log에는 cost_usd가 기록되지만 costs 테이블이 비어있습니다. aiConnector.js에서 추론 완료 후 db.recordCost() 호출 여부를 확인하고, 없다면 추가해야 합니다. (비용 모니터링 핵심)
🔄
어드민 메모리 수치 표시 오류
/api/admin/system 응답에서 memory.total/used/free 값이 0으로 파싱됨. API 응답 구조가 변경됐거나 단위 처리 이슈. UI에서는 pm2.getProcessCPUUsage()로 별도 처리 가능.
🌐
CORS 설정 — 도메인 추가 시 업데이트 필요
현재 ALLOWED_ORIGINS가 비어있어 localhost와 .sandbox.novita.ai만 허용. 실제 도메인 연결 시 .env에 ALLOWED_ORIGINS=https://your-domain.com 추가 필요.
🚨
root 계정으로 PM2 실행 중
현재 PM2가 root 권한으로 실행 중입니다. 보안상 aiorch 사용자로 전환 권장. ecosystem.config.js의 user 설정 및 pm2 startup 재등록 필요.
